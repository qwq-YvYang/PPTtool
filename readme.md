# 🎯 PPT 小工具 — PowerPoint 放映辅助工具

> 一款轻量级的 **PowerPoint 放映辅助工具**，在幻灯片放映时提供浮动工具栏和翻页栏，**完全调用 PowerPoint 原生 API**，无模拟点击，安全稳定。

---

## ✨ 功能一览

| 功能               | 说明                                                         |
| ------------------ | ------------------------------------------------------------ |
| 🖱 **箭头指针**     | 切换为鼠标指针，单击幻灯片任意位置即可翻到下一页             |
| ✏️ **画笔**         | 切换为绘图笔，可在幻灯片上自由标注；再次点击弹出**颜色选择面板** |
| 🧹 **橡皮擦**       | 切换为橡皮擦，擦除手绘笔迹；再次点击显示 **"除该页笔迹"** 一键清除 |
| ⏹ **退出放映**     | 一键退出当前幻灯片放映                                       |
| ◀▶ **翻页栏**      | 左下角和右下角各有一个翻页栏，每个翻页栏都包含 **上一页** 和 **下一页** 按钮 |
| 🔄 **可拖拽**       | 工具栏和翻页栏均可**拖拽移动**到屏幕任意位置                 |
| 🎨 **现代化 UI**    | 白色半透明圆形按钮，精致矢量图标，悬停高亮，背景极简         |
| ⚡ **焦点自动归还** | 点击工具后**焦点自动回到 PowerPoint**，无需二次点击即可操作  |

## 🚀 安装方法

### 系统要求

- **操作系统**：Windows 10 / Windows 11
- **Office**：Microsoft 365 / Office 2021 / Office 2019（32位或64位均可）
- **.NET 运行时**：.NET Framework 4.8（安装包会自动检测安装）

### 安装步骤

1. 下载最新发布的安装包（`setup.exe`）
2. **右键 → 以管理员身份运行**
3. 按照安装向导完成安装
4. **重启 PowerPoint**
5. 打开任意 PPTX 文件，按 **F5** 进入放映模式
6. 底部工具栏和翻页栏会自动出现 🎉

> 💡 **如果安装后未自动加载**：
> PowerPoint → 文件 → 选项 → 加载项 → 管理"COM 加载项" → 转到 → 确保 **PPTTool** 已勾选

### 卸载

控制面板 → 程序和功能 → 找到 "PPTTool" → 卸载

## 🎮 使用指南

### 工具栏（底部居中）

| 按钮             | 操作     | 效果                               |
| ---------------- | -------- | ---------------------------------- |
| 🖱 箭头           | 点击     | 切换为鼠标指针，单击翻页           |
| ✏️ 画笔           | 点击     | 切换为画笔模式，可在幻灯片上绘图   |
| 🧹 橡皮擦         | 点击     | 切换为橡皮擦模式，擦除笔迹         |
| ⏹ 退出           | 点击     | 退出幻灯片放映                     |

### 翻页栏（左下角 + 右下角）

| 按钮     | 操作 | 效果               |
| -------- | ---- | ------------------ |
| ◀ 上一页 | 点击 | 跳转到上一张幻灯片 |
| ▶ 下一页 | 点击 | 跳转到下一张幻灯片 |

### 拖拽移动

- 工具栏和翻页栏的**非按钮区域**可以拖拽移动

- 拖拽到合适位置即可固定

  ---

### 功能实现

- **VSTO Add-in**：嵌入 PowerPoint 进程，完全事件驱动
- **Win32 分层窗口**：实现半透明浮动覆盖层，不阻塞幻灯片操作
- **WM_NCHITTEST**：选择性点击穿透 — 按钮区域可点击，其余区域穿透到 PowerPoint
- **PowerPoint 原生 API**：所有功能（指针切换、画笔颜色、清除笔迹、翻页）均通过 COM 接口调用

## 🔧 自定义开发

### 修改按钮图标

在 `NavOverlay.cs` 中修改以下属性即可自定义图标：

```csharp
// 修改翻页栏图标（任何 Unicode 符号或 Emoji）
public string PrevIcon { get; set; } = "◀";
public string NextIcon { get; set; } = "▶";
```

工具栏图标在 `ToolbarOverlay.cs` 的 `DrawCustomIcon()` 方法中绘制矢量图形，可自由修改。

### 自定义颜色主题
```text
// NavOverlay.cs 中的颜色配置
public Color BorderColor = Color.FromArgb(160, 255, 255, 255);   // 边框
public Color BgColor = Color.FromArgb(30, 255, 255, 255);        // 背景
public Color HoverBorderColor = Color.FromArgb(220, 255, 255, 255); // 悬停边框
public Color IconColor = Color.FromArgb(200, 255, 255, 255);     // 图标颜色
```
